# Puzzle Game (8-Puzzle & 15-Puzzle)

Bienvenue dans le projet **Puzzle Game** ! Ce jeu est développé en Python avec Pygame et peut être exécuté dans un environnement Docker.

## Prérequis

Avant de commencer, assurez-vous d'avoir installé :
- [Docker](https://www.docker.com/)
- [Git](https://git-scm.com/)

## Étapes pour lancer le projet

### 1. Cloner le dépôt
Utilisez la commande suivante pour cloner ce dépôt sur votre machine :
```bash
git clone https://github.com/TatumLn/Puzzles_Game.git
cd Puzzles_Game
